## Installing

1. Clone repo:

```
git clone git@github.com:fckXYZ/mesto.git
cd mesto
```

2. Install debendencies:

```
npm install
```

3. Choose build:
  + `npm run start` - run server at 3000 port.
  + `npm run dev` - run server at 3000 port with hot-reload enabled.
